# 🎁 올해의 결산 도우미 (Red/White Holiday)
매년 올해의 결산 쓰다가, 더 간편하게 쓰는 법 고민해보고 만들어봤어요.
코딩을 제대로 배운 적이 없어서 GPT 사용해서 제작했습니다.

결산 도우미는 질문을 하나씩 던져주고, 답을 모아 한 번에 정리해주는 연말 결산 웹페이지입니다.  
**서버 없이** 단일 `index.html`로 동작하며, 작성 내용은 **브라우저(LocalStorage)에만 저장**됩니다.

## ✨ 주요 기능
- ✅ 질문 순차 진행: **다음 버튼 하나로 진행**
- ✅ 이전 답변 수정: 오른쪽 목록에서 **수정할 질문 클릭 → 해당 질문으로 이동**
- ✅ 결산 결과 복사
  - **번호 리스트로 복사(기본)**
  - 전체 복사
  - 답변한 것만 복사
- ✅ 인스타 **스토리 업로드용 카드(1080×1920 PNG)** 자동 생성

## 🚀 바로 실행하기
### 로컬에서 실행
1. 이 레포를 다운로드(또는 클론)
2. `index.html` 파일을 더블클릭해서 브라우저로 열기

> 크롬/엣지/사파리/파이어폭스 최신 버전 권장

## 🧩 커스터마이즈
- 페이지 안의 `⚙️ 커스터마이즈`에서
  - 제목 변경
  - 질문 목록(줄바꿈 구분) 변경 가능

## 🔒 개인정보/저장
- 이 페이지는 서버로 전송하지 않습니다.
- 작성 내용은 **내 브라우저에만 저장**되며, 다른 기기/브라우저에는 공유되지 않습니다.
